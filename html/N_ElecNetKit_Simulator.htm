ElecNetKit.Simulator NamespaceElecNetKit Documentation
The ElecNetKit.Simulator namespace defines a set of classes for controlling simulation, and managing external simulator integration.
Classes
  ClassDescription
NetworkController
A class that governs the production of a network model, experimenting on the network model and transforming the results (all as requested). Capable of working with any ISimulator, and will automatically detect and use any available simulator in the /ElecNetKitExtensions subfolder of the entry assembly (GetEntryAssembly )'s location.

ISimulator
Defines an electrical network simulator. Simulators must be able to run textual commands and return network models.
