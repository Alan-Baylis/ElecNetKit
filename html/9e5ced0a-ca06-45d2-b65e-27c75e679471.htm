Getting StartedElecNetKit Documentation

This section takes you through some of the basic tasks required to set up and get started with ElecNetKit.

In This Section
  • Setting up a Development Environment covers the installation of Visual Studio and OpenDSS.
  • Walkthrough: A Minimal Working Example walks through the steps required to run, output and graph the results of a simple network experiment.
  • Traversing the Network Model covers the exploration of ElecNetKit electrical network models, including topology traversal (what's connected to what), and obtaining characteristics and results.
