Remarks
The Phased T  interface is intended to be used directly by all
            client code. Implementors of NetworkElements should consider using the
            types PhasedValues T  and PhasedEvaluated TFrom, TTo , which implement
            the Phased T  interface with a set of values and an evaluator function
            respectively. For example usage, see the implementation of Bus.
