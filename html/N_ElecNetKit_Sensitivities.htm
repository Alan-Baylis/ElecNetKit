ElecNetKit.Sensitivities NamespaceElecNetKit Documentation
The ElecNetKit.Sensitivities namespace defines a set of classes for running perturb-and-observe style experiments, and for obtaining sensitivity information from them.
Classes
  ClassDescription
ObserveResultsTransform T 
A transform that observes and saves data from specific network elements.
PerturbAndObserveRunner TObserve 
Runs perturb-and-observe experiments on electrical network models.
SensitivityGenerator T 
Generates sensitivity information from the values obtained from applying a PerturbAndObserveRunner TObserve  to an electrical network model.
VoltageSensitivityToComplexPowerGenerator
Generates Voltage Sensitivities to changes in P and Q.
VoltageSensitivityToPQDataSet
Holds a set of sensitivities between two buses.
