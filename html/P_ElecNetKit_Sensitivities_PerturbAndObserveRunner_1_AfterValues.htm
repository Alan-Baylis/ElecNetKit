Field Value

            The keys of the outer dictionary provided by this property are Tuple T1, T2 s, where
            Item1 is a Tuple T1, T2  specifying the ID and type of the perturbing network element, and Item2
            is the set of perturbation values, as specified by PerturbValuesToRecord.
            The keys of the inner dictionary are Tuple T1, T2 s specifying the ID and type of the observed network element.
            The values of the inner dictionary are the values required to be observed by ObserveElementValuesSelector.
