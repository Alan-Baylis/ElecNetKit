Remarks
When displaying graphs from a console application, you can use
            StartGraphHostWindow(INetworkGraph) instead of instantiating a new
            GraphHostWindow.
            Alternatively, if you require a reference to a GraphHostWindow,
            you can mark the application entry point (usually Main()) with
            STAThreadAttribute. See The calling thread must be STA for more information.
